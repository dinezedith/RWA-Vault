# RWA Vault Client

This is the client application for **RWA Vault**.

## Getting Started

1. **Install dependencies:**
    ```bash
    npm install
    ```
2. **Run the development server:**
    ```bash
    npm start
    ```

## Project Structure

- `src/` - Source code
- `public/` - Static assets

## Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm test` - Run tests
